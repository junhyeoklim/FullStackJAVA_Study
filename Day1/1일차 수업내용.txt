- 수업 내용 -
*javac (자바파일이름).java : 자바 컴파일 (클래스 파일 생성)
**java (클래스 파일 이름) : 도스에서 자바 파일 실행

1) static : 객체 생성 없이 한다?(기억이 맞다면 메모리에 미리 올려놓는것을 뜻함)
	 JVM이 main함수를 곧바로 실행할 수 있도록 한다.

2)  String[] args : 커맨드라인 등을 통해, main함수 내부에서 사용할 수 있는 
	          String 데이터를 전달할 수 있다.

3) println(5+3+"안녕하세요") vs println("안녕하세요"+5+3) 차이 :
   전자는 int자료형으로 시작 하기 때문에 출력 결과가 '8안녕하세요'로 나오지만
   후자는 문자열 "안녕하세요"랑 int형 자료형인 5랑 서로 데이터 타입이 다르기 때문에 
   5가 문자열로 인식되고 마찬가지로 3이 문자열로 형 변화가 일어나
   출력 결과가 '안녕하세요53'가 된다.

4) while 반복문 쓸 때 조심해야할 점 : 조건을 만족하여 반복문을 빠져 나가기 전에 증감이 1번 더 된다
